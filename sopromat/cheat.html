⚡ Шпаргалка перед экзаменом

Сопротивление материалов · формулы, знаки и ловушки · печать: 2–3 страницы A4

🧮 Все формулы

Сначала выбери вид нагружения, затем формулу. A — площадь сечения (в некоторых заданиях обозначена F), J — момент инерции, W — момент сопротивления.

Растяжение и сжатие

  • σ = N/A
  • ε = Δl/l
  • σ = Eε
  • Δl = Nl/(EA); для ступенчатого стержня Δl = ΣNᵢlᵢ/(EᵢAᵢ)
  • Продольная жёсткость: EA
  • μ = |ε⊥/ε∥|
  • n = σпред/|σ|; [σ] = σпред/n
  • Прочность: |N|max/A ≤ [σ]
  • δ = (lпосле − l₀)/l₀ · 100%
  • ψ = (A₀ − Aшейки)/A₀ · 100%

Сдвиг и кручение

  • τ = Gγ
  • ΔS = Ql/(GA)
  • G = E/[2(1 + μ)]
  • τ = Mк·ρ/Jp
  • τmax = |Mк|max/Wp
  • Wp = Jp/R
  • φ = Mкl/(GJp); общий случай φ = ∫Mк/(GJp) dx
  • Жёсткость при кручении: GJp
  • Прочность: |Mк|max/Wp ≤ [τ]
  • Жёсткость: φmax ≤ [φ]
  • M = P/ω; при P в кВт и n в об/мин: M ≈ 9550P/n, Н·м

Напряжённое состояние

  • Полное напряжение: p = √(σ² + τ²)
  • Главные напряжения: σ₁ ≥ σ₂ ≥ σ₃ алгебраически; на главной площадке τ = 0
  • Объёмное состояние: ε₁ = [σ₁ − μ(σ₂ + σ₃)]/E и ещё две циклические формулы
  • Плоское состояние: ε₁ = (σ₁ − μσ₂)/E, ε₂ = (σ₂ − μσ₁)/E, ε₃ = −μ(σ₁ + σ₂)/E
  • Линейное состояние: ε₁ = σ₁/E, ε₂ = ε₃ = −μσ₁/E
  • Инвариант: σα + σβ + σγ = σ₁ + σ₂ + σ₃; в плоскости σα + σβ = σ₁ + σ₂
  • Одноосное растяжение: σα = σ₀cos²α
  • τmax = (σmax − σmin)/2
  • K = E/[3(1 − 2μ)]; εV = ε₁ + ε₂ + ε₃ = (σ₁ + σ₂ + σ₃)/(3K)
  • εV = 0 при μ = 0,5 или σ₁ + σ₂ + σ₃ = 0
  • III теория: σэкв = √(σ² + 4τ²); IV теория: σэкв = √(σ² + 3τ²)
  • Вал: Mэкв,III = √(Ми² + Мк²); Mэкв,IV = √(Ми² + 0,75Мк²)
  • После момента: σэкв = Mэкв/W ≤ [σ]

Геометрические характеристики

  • Sx = ∫y dA; Sy = ∫x dA
  • Jx = ∫y²dA; Jy = ∫x²dA; Jxy = ∫xy dA
  • Jp = ∫ρ²dA = Jx + Jy
  • W = J/ymax; Wp = Jp/R
  • Прямоугольник: J = bh³/12, W = bh²/6; ось параллельна стороне b
  • Круг: J = πd⁴/64 ≈ 0,05d⁴, W = πd³/32 ≈ 0,10d³
  • Круг: Jp = πd⁴/32 ≈ 0,10d⁴, Wp = πd³/16 ≈ 0,20d³

Изгиб

  • σ = My/J
  • σmax = M/W — в проверке прочности берут |M|max/W ≤ [σ]
  • Касательные напряжения: τ = QS/(Jb), где b — местная ширина сечения
  • Q = dM/dx
  • В принятой здесь знаковой конвенции: q = dQ/dx = d²M/dx²
  • EJy″ = M; одно интегрирование даёт угол y′, два — прогиб y
  • Заделка: y = 0, y′ = 0; шарнир: y = 0
  • Чистый изгиб: Q = 0, M = const

О знаках: если в другой конвенции распределённую нагрузку, направленную вниз, считают положительной, в зависимостях для q появляется минус. В задачах сначала смотри принятое направление осей и знаков.

📊 Правила знаков и эпюр

  • N > 0, σ > 0 — растяжение; N < 0, σ < 0 — сжатие. Положительное нормальное напряжение направлено от сечения по внешней нормали.
  • Q > 0, если сила направлена вверх на левой отсечённой части или вниз на правой.
  • M > 0, если балка изгибается выпуклостью вниз.
  • Касательное напряжение положительно, если поворот его вектора против часовой стрелки на π/2 совмещает его с внешней нормалью.
  • Закон парности: на взаимно перпендикулярных площадках касательные напряжения равны по модулю. В принятой в задачах скалярной записи τβ = −τα, поэтому +50 → −50 МПа; физически знак зависит от ориентации площадок.
  • Сосредоточенная продольная сила даёт скачок N; поперечная сила или реакция — скачок Q.
  • Сосредоточенный момент даёт скачок M, но никак не отмечается на эпюре Q.
  • Если q=0, то Q=const, а M меняется линейно. При q=const эпюра Q линейна, эпюра M — парабола. В записи q=dQ/dx q алгебраическая: направленная вниз нагрузка отрицательна.
  • На концевом шарнире или свободном конце M=0, только если там не приложена сосредоточенная пара сил.
  • При постоянных Mк, G, Jp угол закручивания меняется по участку линейно.

📐 СИ и числа-маяки

Переводы и размерности

  • 1 кН = 10³ Н; 1 кН·м = 10³ Н·м
  • 1 МПа = 10⁶ Па = 1 Н/мм²; 1 ГПа = 10³ МПа
  • 1 см² = 10⁻⁴ м²; 1 мм² = 10⁻⁶ м²
  • 1 см³ = 10⁻⁶ м³; 1 мм³ = 10⁻⁹ м³
  • 1 см⁴ = 10⁻⁸ м⁴; 1 мм⁴ = 10⁻¹² м⁴
  • P, N, Q — Н; M — Н·м; q — Н/м.
  • σ, τ, E, G — Па; A — м²; J — м⁴; W — м³.
  • ε, γ, μ безразмерны; температурный коэффициент — °C⁻¹.

Увидел числа — вспомни ход

  • 10 МПа · 225 см² → 225 кН
  • 20·10⁶ Н / 2 м² → 10 МПа, но 20 Н / 2 м² → 10 Па
  • Консоль: 5 кН, 1 м, b=6 см, h=10 см → 50 МПа
  • σmax=80 МПа, σm=20 МПа → σa=60 МПа
  • 30 + 40 → 70 МПа по инварианту
  • σ=4 МПа, τ=3 МПа → p=5 МПа
  • 5M − 2M при M=20 кН·м → 60 кН·м
  • 0,3/150 → ε=2·10⁻³
  • Внезапная нагрузка: Kд=2
  • Длина стержня ×3 → Pкр ÷9
  • Диаметр вала ×2 → τmax ÷8; диаметр ÷3 → τmax ×27
  • На нейтральной оси балки и на оси круглого вала соответствующее напряжение равно 0.

⚠️ Топ-10 ловушек

  1. Сен-Венан: достаточно далеко от места приложения нагрузки напряжения и деформации не зависят от способа её приложения.
  2. Пределы: пропорциональности → упругости → текучести → прочности. Условный предел σ₀,₂ — это остаточная пластическая деформация 0,2%.
  3. Материал: допускаемое напряжение пластичного материала назначают по σт, хрупкого — по σв.
  4. Доли и проценты: ε — доля; δ и ψ обычно выражают в процентах. Не теряй множитель 100%.
  5. Степени длины: A — м², W — м³, J — м⁴. В bh³/12 в кубе размер, перпендикулярный оси.
  6. Опоры: подвижный шарнир / неподвижный шарнир / заделка дают соответственно 1 / 2 / 3 реакции.
  7. Кручение: τmax обратно пропорционально d³; на оси τ=0, на поверхности максимум.
  8. Главные напряжения: упорядочивай алгебраически, не по модулю. Для произвольной площадки σ₂ ≤ σα ≤ σ₁; строгие знаки — только если площадка не главная.
  9. Два закона Гука: в плоском состоянии два ненулевых главных напряжения, в объёмном — три. Не выбрасывай σ₃ без основания.
  10. Эпюры: сила скачком меняет Q, момент — M; чистый изгиб имеет Q=0. В теориях прочности: III содержит 4τ², IV — 3τ².

🩶 Чего скорее всего не будет

Это низкий приоритет, а не нулевая вероятность. Сначала закрепи основные шесть тем, затем просмотри этот блок для узнавания формулировок.

Устойчивость

  • Pкр = π²EJmin/(μl)²
  • σкр = π²E/λ²
  • Эйлер — большая гибкость: λ ≥ λпред, эквивалентно σкр ≤ σпц.
  • Ясинский — средняя гибкость: σкр = a − bλ, Pкр = (a − bλ)A.
  • Малая гибкость — расчёт как на сжатие, приближённо Pкр = σтA; с учётом продольного изгиба P/A ≤ [σ]у = φ[σ].
  • Увеличение длины в три раза уменьшает силу Эйлера в девять раз.

Усталость и динамика

  • σm = (σmax + σmin)/2; σa = (σmax − σmin)/2
  • R = σmin/σmax; симметричный цикл: R = −1, предел обозначают σ₋₁.
  • Удар: Kд = 1 + √(1 + 2h/Δст); внезапное приложение h=0 ⇒ Kд=2.
  • Гармоническая нагрузка без демпфирования: Kд = 1/|1 − (ω/ω₀)²|; коэффициент может быть больше или меньше единицы.

Сложное сопротивление

  • Внецентренное сжатие: σ = −P(1/A + yp·y/Jz + zp·z/Jy); возникают N, My, Mz и только нормальные напряжения.
  • Косой изгиб: σ = Mz·y/Jz + My·z/Jy; f = √(fy² + fz²).
  • Ядро сечения — область приложения силы, при которой во всём сечении напряжения сохраняют один знак.

Остальное

  • II теория прочности: εmax ≤ [ε].
  • Степень статической неопределимости = число независимых неизвестных минус число независимых уравнений равновесия; основной способ раскрытия — метод сил.
  • Собственный вес: P/A + γl ≤ [σ]; брус равного сопротивления: A(x) = A₀exp(γx/[σ]).
  • Температурный коэффициент имеет размерность °C⁻¹.
  • Также достаточно узнавать концентрацию напряжений, теории прочности, принцип Даламбера и реакции гибких связей.
