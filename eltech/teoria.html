📚 Теория по полочкам

Всё, что нужно для твоего экзамена — по темам реальных вопросов. Открывай раздел, читай, потом прогоняй тему в тренажёре.

0. Стратегия: вечер перед экзаменом
  1. Сейчас: прочитай разделы 1–4 (час) → в тренажёре режим «Заучивание», темы «Постоянный ток» и «Переменный ток».
  2. Потом: разделы 5–8 (40 мин) → заучивание остальных тем.
  3. Перед сном: один билет в режиме 🎓 Экзамен. Ошибки — в «Разбор», читай пояснения.
  4. Утром: «Разбор» → сортировка «сначала мои ошибки» → пробеги свои слабые места. Ещё один билет, если есть время.
В тренажёре у каждого вопроса есть пояснение с ходом решения — это твой главный «решебник». Теория ниже объясняет, ПОЧЕМУ так.

3. Словарь: узел, ветвь, контур, цепь
  • Электрическая цепь — совокупность элементов, образующих путь для тока, процессы в которых описываются понятиями ЭДС, тока и напряжения.
  • Узел — точка, где соединяются три и более ветви.
  • Ветвь — участок между двумя узлами, по которому течёт один и тот же ток.
  • Контур — замкнутый путь по ветвям, где узлы не повторяются.
  • Параллельные ветви — присоединены к одной паре узлов.
  • ВАХ — вольт-амперная характеристика: зависимость между напряжением и током элемента. Линейный элемент — ВАХ прямая; нелинейный — ВАХ кривая.

Сколько уравнений составлять?

Узлов У, ветвей В: по 1-му закону Кирхгофа — У−1 уравнений, по 2-му — В−У+1 (это же число независимых контуров и контурных токов).

4. Постоянный ток — фундамент (учи первым!)

Три главных закона

  • Закон Ома: I = U / R (запомни и производные: U = I·R, R = U/I).
  • 1-й закон Кирхгофа (про УЗЕЛ и ТОКИ): алгебраическая сумма токов, сходящихся в узле, равна нулю (сколько втекло — столько вытекло).
  • 2-й закон Кирхгофа (про КОНТУР и НАПРЯЖЕНИЯ): алгебраическая сумма падений напряжения на пассивных элементах замкнутого контура равна алгебраической сумме ЭДС этого контура: Σ I·R = Σ E.
⚠️ Как отличить в тесте: «сумма ТОКОВ… в УЗЛЕ» → 1-й закон. «Сумма падений НАПРЯЖЕНИЯ… в КОНТУРЕ = сумме ЭДС» → 2-й закон. «Связь тока и напряжения на участке» → закон Ома.

Соединения резисторов

  • Последовательно: R = R₁ + R₂ + … (ток общий).
  • Параллельно: R = R₁·R₂/(R₁+R₂) для двух; N одинаковых R → R/N.
  • Смешанное: сворачивай с дальнего конца схемы к источнику, шаг за шагом.

Метод наложения — пошагово

1) Оставь в схеме ОДИН источник. Остальные «убей»: источник ЭДС → замени перемычкой (КЗ), источник тока → разрыв.
2) Посчитай частичный ток от этого источника (обычно простой закон Ома + делитель).
3) Повтори для каждого источника. Итоговый ток = алгебраическая сумма частичных: I₁ = I₁′ + I₁″.
⚠️ Типовой вопрос: «ток I₁′ в отсутствие источника тока» — убери J (разрыв!), останется один контур с E: I₁′ = E / ΣR по контуру.

Метод контурных токов (МКТ)

1) Выбери независимые контуры (их В−У+1), каждому — свой контурный ток Iк.
2) Для контура: (сумма ВСЕХ R контура)·Iк(свой) ± Rсмежных·Iк(соседних) = ±ΣE контура. Знак «+» у соседнего, если токи в смежной ветви сонаправлены.
3) Настоящий ток ветви = сумма/разность контурных токов, проходящих через неё.

Мощность и баланс

  • P = U·I = I²·R = U²/R — все три формы!
  • Баланс: мощность, отданная источниками (ΣE·I и ΣUJ·J) = мощность потребителей (ΣI²R).
  • Энергия: W = P·t = I²·R·t (закон Джоуля–Ленца — про НАГРЕВ).
5. Переменный ток — самая большая тема твоего экзамена

Читаем запись u(t) = Um·sin(ωt + ψ)

Часть записиЧто это
Umамплитуда (максимум)
U = Um/√2действующее значение (именно его показывает вольтметр!). Если написано 220√2·sin(…) — действующее 220 В
ωугловая частота, рад/с. f = ω/2π. Запомни: ω=314 → f=50 Гц; ω=628 → 100 Гц

Кто отстаёт, кто опережает (векторные диаграммы)

ЭлементСдвиг фаз
Резистор Rток и напряжение В ФАЗЕ (φ=0)
Катушка Lток ОТСТАЁТ от напряжения на 90° (u опережает i)
Конденсатор Cток ОПЕРЕЖАЕТ напряжение на 90°
Мнемоника: в слове «КЛюч» — К (конденсатор) ток впереди; у катушки (L) ток Ленивый — отстаёт.

Сопротивления

  • Индуктивное: XL = ωL = 2πfL (растёт с частотой).
  • Ёмкостное: XC = 1/(ωC) (падает с частотой).
  • Полное (последовательная RLC): Z = √(R² + (XL−XC)²).
  • Закон Ома: I = U/Z.

Комплексные числа — рецепт перевода

Показательная → алгебраическая: A·ejφ = A·cosφ + j·A·sinφ. Пример: 100·ej45° = 70,7 + j70,7.
Алгебраическая → показательная: модуль A = √(a²+b²), угол φ = arctg(b/a). Пример: 30+j40 → 50·ej53°.
Комплексное сопротивление: Z = R + j(XL−XC). Комплекс тока из синусоиды i(t)=Imsin(ωt+ψ): действующее Im/√2, угол ψ.

Характер нагрузки по углу Z — очень частый вопрос!

Вид ZХарактер
Z = R (угол 0°)активный
+j… (угол 0…+90°)активно-индуктивный
−j… (угол 0…−90°)активно-ёмкостный (пример: 500·e−j45°)

Мощности

  • Активная: P = U·I·cosφ [Вт] — греет резисторы: P = I²·R.
  • Реактивная: Q = U·I·sinφ [ВАр].
  • Полная: S = U·I [ВА]; треугольник мощностей: S² = P² + Q².
  • Комплексная: S̃ = Ũ·Ĩ* (ток с сопряжённым знаком угла!). Угол S̃ = φu − φi.
  • cosφ = P/S — коэффициент мощности.

Резонанс

  • Условие: XL = XC → частота ω₀ = 1/√(LC).
  • При резонансе: Z = R (минимум для последовательной цепи), ток максимален, цепь ведёт себя как чисто активная, cosφ=1.
  • Резонанс напряжений — в ПОСЛЕДОВАТЕЛЬНОЙ RLC (UL и UC могут превышать входное!). Резонанс токов — в ПАРАЛЛЕЛЬНОЙ.
  • Добротность Qд = Q/P (отношение реактивной мощности к активной) = UL/U при резонансе.
6. Трёхфазные цепи

Главные соотношения — просто заучи

СхемаНапряженияТоки
Звезда (Y)Uл = √3·UфIл = Iф
Треугольник (Δ)Uл = UфIл = √3·Iф
  • √3 ≈ 1,73. Стандарт сети: фазное 220 В ↔ линейное 380 В (√3·220).
  • Линейное напряжение — между двумя фазами (A–B); фазное — между фазой и нейтралью (A–N).
  • Сдвиг между фазами — 120°. Прямое чередование: A→B→C.
  • Симметричная нагрузка (все Z одинаковы): ток в нейтральном проводе = 0, нейтраль можно убрать.
  • Несимметричная нагрузка без нейтрали → перекос фазных напряжений (нейтраль выравнивает).
  • Мощность: P = 3·Uф·Iф·cosφ = √3·Uл·Iл·cosφ. На ОДНУ фазу: P1ф = Uф·Iф·cosφ.
⚠️ Вопрос «отношение линейного тока к фазному при треугольнике» → √3. «При звезде» → 1. Не перепутай, где √3 у токов, а где у напряжений!
7. Магнитные цепи и трансформатор
  • Магнитная цепь — совокупность магнитных И немагнитных участков, по которым замыкается магнитный поток.
  • Потокосцепление Ψ = w·Φ [Вб] (w — число витков).
  • Индуктивность L = Ψ/i [Гн].
  • Закон электромагнитной индукции: e = −dΨ/dt — ЭДС наводится при ИЗМЕНЕНИИ потока.
  • ЭДС самоиндукции: eL = −L·di/dt. Напряжение на катушке: uL = L·di/dt. На конденсаторе: iC = C·du/dt (не перепутай, кто где!).
  • Ферромагнетики: петля гистерезиса, насыщение; потери в сердечнике = гистерезис + вихревые токи (поэтому сердечник из изолированных листов).

Трансформатор

  • Работает только на переменном токе (нужен изменяющийся поток!).
  • Принцип: закон электромагнитной индукции — переменный поток в сердечнике наводит ЭДС во вторичной обмотке.
  • Коэффициент трансформации: k = w₁/w₂ = U₁/U₂ ≈ I₂/I₁. k>1 — понижающий.
  • Режимы: холостой ход (вторичка разомкнута — измеряют k и потери в стали), короткое замыкание (опыт — потери в меди), нагрузочный.
8. Электрические машины
  • Частота вращения магнитного поля: n₀ = 60·f/p об/мин (p — число пар полюсов). f=50, p=1 → 3000; p=2 → 1500.
  • Асинхронный двигатель: ротор вращается МЕДЛЕННЕЕ поля. Скольжение s = (n₀−n)/n₀ (обычно 2–8%). Простой, дешёвый, самый массовый.
  • Синхронный: ротор вращается ТОЧНО с полем (s=0); используется и как генератор на электростанциях.
  • Машина постоянного тока: есть коллектор (щётки); регулируется скорость просто; обратима (мотор↔генератор).
  • Обратимость: любая машина может работать и двигателем, и генератором.
9. Электроника — диоды и транзисторы
  • p-n переход проводит ток в одну сторону (прямое включение: «+» к p-области).
  • Диод — один p-n переход; применение — выпрямление переменного тока. ВАХ несимметричная: прямая ветвь круто вверх, обратная — почти ноль до пробоя.
  • Стабилитрон — работает на ОБРАТНОЙ ветви (пробой), держит постоянное напряжение — стабилизация.
  • Выпрямители: однополупериодный — 1 диод (пропускает половину волны); двухполупериодный мост — 4 диода, использует обе полуволны; для трёхфазного моста — 6.
  • Транзистор (биполярный: эмиттер-база-коллектор) — усиление сигналов, электронный ключ. Маленький ток базы управляет большим током коллектора.
  • Тиристор — управляемый диод: открывается импульсом на управляющий электрод; управляемые выпрямители.
  • Варикап — ёмкость зависит от обратного напряжения; варистор — нелинейный резистор U(I).
  • Фотодиод/светодиод: свет→ток / ток→свет.
10. Переходные процессы — 4 правила решают всё

Законы коммутации (что НЕ меняется скачком)

  • 1-й: ток через ИНДУКТИВНОСТЬ не может измениться скачком: iL(0⁺) = iL(0⁻).
  • 2-й: напряжение на КОНДЕНСАТОРЕ не может измениться скачком: uC(0⁺) = uC(0⁻).

Как ведут себя L и C в предельных режимах (магия для тестов)

МоментКатушка L (если ток был 0)Конденсатор C (если был разряжен)
Сразу после включения (t=0⁺)РАЗРЫВ (тока нет)ПЕРЕМЫЧКА-КЗ (напряжения нет)
Установившийся режим (t→∞), пост. токПЕРЕМЫЧКА-КЗ (XL=0)РАЗРЫВ (ток не течёт)
  • Постоянная времени: RC-цепь τ = R·C; RL-цепь τ = L/R. За τ процесс проходит ~63%, за 3–5τ — практически весь.
  • Все токи/напряжения меняются по экспоненте e−t/τ.
11. Измерения
  • Амперметр — ПОСЛЕДОВАТЕЛЬНО с нагрузкой, Rвн→0. Расширение предела — шунт (резистор параллельно прибору).
  • Вольтметр — ПАРАЛЛЕЛЬНО, Rвн→∞. Расширение предела — добавочное сопротивление (последовательно).
  • Ваттметр — две катушки: токовая последовательно + катушка напряжения параллельно; показывает активную мощность P.
  • Вольтметр в цепи переменного тока показывает действующее значение.
  • Счётчик энергии — кВт·ч.
12. Шпаргалка-выжимка на утро (прочитай за 10 минут перед выходом)
  • I=U/R · P=UI=I²R · последовательное складывается, параллельное: R₁R₂/(R₁+R₂)

  • Наложение: E→КЗ, J→разрыв
  • Идеальные: E: R=0 · J: R=∞ · амперметр: R=0 · вольтметр: R=∞

  • −j → ёмкостный, +j → индуктивный, чистое число → активный
  • R: в фазе · L: ток отстаёт на 90° · C: ток опережает на 90°
  • S²=P²+Q² · P=UIcosφ [Вт] · Q [ВАр] · S [ВА]
  • Резонанс: XL=XC, cosφ=1, ток максимален (посл. цепь)
  • Звезда: Uл=√3Uф, токи равны · Треугольник: Iл=√3Iф, напряжения равны · 380=√3·220 · сдвиг фаз 120°
  • Симметричная нагрузка → ток нейтрали 0
  • Трансформатор: k=w₁/w₂=U₁/U₂, только переменный ток
  • Коммутация: iL и uC не скачут · τ=RC или L/R
  • t=0: L-разрыв, C-КЗ · t=∞: L-КЗ, C-разрыв
  • Диод — выпрямление · стабилитрон — стабилизация · транзистор — усиление · мост — 4 диода
